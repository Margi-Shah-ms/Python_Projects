# 🚀 Logic Box – Pattern Generator & Number Analyzer

> **A Python console application that combines pattern generation and number analysis to demonstrate the use of loops, functions, conditional statements, and menu-driven programming.**

---

# 📖 About the Project

**Logic Box** is a Python console application developed to strengthen fundamental programming concepts through two interactive modules.

The application provides a simple **menu-driven interface** where users can choose between generating a star pattern, analyzing a range of numbers, or exiting the program. It demonstrates structured programming while keeping the code organized and easy to understand.

---

# 🎯 Project Objectives

The main objectives of this project are to:

* Practice Python programming fundamentals
* Implement functions and modular programming
* Use loops and nested loops effectively
* Apply conditional statements for decision-making
* Handle user input and validation
* Build a menu-driven console application
* Improve logical thinking and problem-solving skills

---

# ✨ Features

## 🎨 Pattern Generator

Generates a **right-angled triangle** based on the number of rows entered by the user.

### Sample Output

```text
*
**
***
****
*****
```

---

## 🔢 Number Analyzer

Analyzes numbers within a user-defined range.

The program will:

* Determine whether each number is **Odd** or **Even**
* Calculate the **sum of all numbers** in the selected range
* Display the results clearly

# ✅ Input Validation

The application validates user input by:

* Ensuring the number of rows is greater than zero.
* Checking that the ending value is not less than the starting value.
* Displaying appropriate error messages for invalid input.
* Allowing users to continue using the application without restarting it.
