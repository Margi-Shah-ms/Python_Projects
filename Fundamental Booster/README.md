# 🚀 Interactive Personal Data Collector

## 📌 Project Overview

The **Interactive Personal Data Collector** is a Python application that collects personal information from a user, processes the data, and displays the results in a user-friendly format.

This project demonstrates fundamental Python programming concepts such as:

* 🖨️ `print()`
* ⌨️ `input()`
* 📦 Variables
* 🔢 Data Types
* 🔄 Type Casting
* ➖ Arithmetic Operators
* 📝 String Formatting
* 🏷️ `type()`
* 🧠 `id()`

The program also calculates the user's approximate birth year based on their age.

---

## 🎯 Objective

The objective of this project is to:

* Collect personal information from the user.
* Process the entered data.
* Demonstrate Python fundamentals.
* Display organized and readable output.

---

## ✨ Features

### 👋 Welcome Message

Displays a greeting message when the program starts.

```text
Welcome to the Interactive Personal Data Collector!
```

### 📝 User Input Collection

The program collects:

* 👤 Name (String)
* 🎂 Age (Integer)
* 📏 Height in meters (Float)
* ⭐ Favourite Number (Integer)

---

## ⚙️ Data Processing

The program:

* Stores user information in variables.
* Converts data into appropriate data types.
* Calculates the approximate birth year.

Formula used:

```python
Birth_Year = Current_Year - age
```

## 🧪 Python Concepts Used

| Concept               | Purpose                             |
| --------------------- | ----------------------------------- |
| 🖨️ print()           | Display output                      |
| ⌨️ input()            | Collect user input                  |
| 📦 Variables          | Store information                   |

## 🔄 Program Flow

1. 👋 Display welcome message
2. ⌨️ Ask the user for personal information
3. 📦 Store data in variables
4. 🏷️ Display data types and memory addresses
5. 🎂 Calculate approximate birth year
6. 📋 Display final information
7. 🙏 Display goodbye message
