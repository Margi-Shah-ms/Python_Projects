# 🍬 Functional Treat

## 📖 Overview

Functional Treat is a menu-driven Python application designed to demonstrate fundamental and intermediate programming concepts through data analysis and transformation tasks.
The project enables users to input numerical data and perform operations such as statistical analysis, recursive calculations, filtering, sorting, and dataset summarization.

The application showcases the practical implementation of Python functions, recursion, lambda expressions, and list manipulation techniques.

---

## 🎯 Objectives

- Analyze and transform user-provided datasets.
- Demonstrate the use of Python built-in functions.
- Implement user-defined functions for custom data operations.
- Apply recursion to solve mathematical problems.
- Utilize lambda functions for data filtering.
- Return multiple values from functions.
- Perform sorting and statistical analysis on datasets.
- Provide an interactive menu-driven user experience.

---

## ✨ Features

### 📊 Dataset Management
- Accept numerical data from the user.
- Store and manage dataset values using Python lists.

### 📈 Data Summary
Generate a quick overview of the dataset using:
- `len()`
- `sum()`
- `min()`
- `max()`

### 🛠️ User-Defined Functions
- Calculate average values.
- Identify duplicate entries.
- Extract unique values.

### 🔄 Recursive Operations
- Factorial Calculation
- Fibonacci Sequence Generation

### 🎛️ Lambda-Based Filtering
- Filter values based on user-defined thresholds.
- Perform simple data transformations.

### 🔃 Sorting Operations
- Ascending Order
- Descending Order

### 📉 Statistical Analysis
Display:
- Minimum Value
- Maximum Value
- Average Value
