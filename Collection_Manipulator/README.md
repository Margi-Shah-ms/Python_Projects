# 🎓 Student Data Organizer

A Python-based **Student Data Organizer** that allows users to efficiently manage student records through a menu-driven interface. This project demonstrates the use of Python's core programming concepts such as **strings, lists, dictionaries, tuples, sets, mutability, immutability, type casting, and the `del` keyword**.

---

## 📌 Project Objective

The objective of this project is to create a collection-based student data management system that allows users to:

- Add new student records
- Display all student records
- Update existing student information
- Delete student records
- Display all unique subjects offered by students

This project was developed as part of a Python programming assignment to practice collection data types and data manipulation.

---

## ✨ Features

- ➕ Add new student records
- 📋 Display all stored student information
- ✏️ Update existing student details
- ❌ Delete student records using Student ID
- 📚 Display unique subjects using Python Sets
- 📅 Store Date of Birth using Tuples
- 🔍 Menu-driven interface for easy navigation
- 🖥️ User-friendly formatted output

## 📂 Data Structure Used

Each student record contains:

| Field | Data Type |
|--------|----------|
| Student ID | Integer |
| Name | String |
| Age | Integer |
| Grade | String |
| Date of Birth | Tuple |
| Subjects | Set |

All student records are stored inside a **List**, where each student is represented as a **Dictionary**.

Example:

```python
{
    "id": 101,
    "name": "Alice",
    "age": 20,
    "grade": "B+",
    "dob": (2002, 5, 14),
    "subjects": {"Math", "Science", "English"}
}
```
